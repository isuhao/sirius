# sirius

## Introduction

Duplicate Checking for Papers(Title and Abstract).

## Feature 

+ use [CppJieba] as chinese word segmenter.
+ support `utf-8` encoding.

## Usage

```
mkdir build
cd build
cmake ..
make
```
